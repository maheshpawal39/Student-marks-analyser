# 📊 Student Marks Analyser

A simple Python project that analyzes student marks using basic data analysis techniques. It calculates key statistics, identifies top performers, and provides insights from the dataset.

## 🚀 Features
- Analyze student marks
- Calculate average, highest, and lowest scores
- Identify top-performing students
- Simple and beginner-friendly code

## 🛠️ Tech Stack
- Python
- Jupyter Notebook
- Pandas
- NumPy

## ▶️ How to Run
1. Clone this repository.
2. Open `student_marks_analyser.ipynb` in Jupyter Notebook.
3. Run all cells.
